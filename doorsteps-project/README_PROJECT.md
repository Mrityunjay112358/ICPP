# 🚀 The Doorsteps Project - React App

## **INSANE FEATURES:**
✨ Built with React + Vite + GSAP  
🎨 Apple-level design & animations  
🎭 Custom cursor with magnetic effects  
🌊 Smooth scroll & parallax effects  
💫 Page transitions & micro-interactions  
📱 Fully responsive design  

## **Tech Stack:**
- **React 18** - Modern React with hooks
- **Vite** - Lightning-fast build tool  
- **GSAP 3** - Professional-grade animations
- **React Router** - Smooth page transitions
- **Framer Motion** - Additional animation library

## **Getting Started:**

```bash
# Navigate to project
cd doorsteps-project

# Install dependencies (already done)
npm install

# Run development server  
npm run dev

# Build for production
npm run build
```

## **Open the app:**
http://localhost:5173
